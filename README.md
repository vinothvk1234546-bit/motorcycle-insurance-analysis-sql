🏍️ MOTORCYCLE INSURANCE POLICY ANALYSIS USING SQL
📌 Project Overview

This project analyzes motorcycle insurance policy data using SQL to identify risk patterns, evaluate claim performance, and generate business insights. The analysis focuses on premium growth trends, claim settlement ratios, and high-risk customer segments.

The objective is to simulate a real-world insurance analytics scenario and derive actionable insights using structured query language.

🎯 Business Problem

Insurance companies need to:

Monitor claim settlement performance

Identify high-risk customers and regions

Track premium revenue growth

Evaluate profitability through claim-to-premium ratios

This project answers these business questions using SQL-based analysis.
